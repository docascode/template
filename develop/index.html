Getting Started Download

Generate static sites from Markdown and code files

DocFX can produce documentation from source code (including C#, F#, Visual Basic, REST, JavaScript, Java, Python and TypeScript) as well as raw Markdown files.

Run everywhere

DocFX can run on Linux, macOS, and Windows. The generated static website can be deployed to any host such as GitHub Pages or Azure Websites with no additional configuration.

Easily customize the output

DocFX provides a flexible way to customize templates and themes.
